# Web-based AI PDF Agent

A web-based AI agent that accepts multiple PDFs, extracts essential information using AI, and outputs structured JSON.

## Features
- Multiple PDF upload
- OCR for scanned PDFs
- Automatic document understanding
- JSON output
- Frontend + Backend

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- Gemini AI
- Tesseract OCR

## How to Run
1. Run backend using Flask
2. Open frontend index.html in browser
3. Upload PDF and extract data

## API Key
Add your Gemini API key in backend/ai_agent.py
